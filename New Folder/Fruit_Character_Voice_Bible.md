# FRUIT CARTOON CHANNEL — CHARACTER AUDIO/VOICE BIBLE
### (Third companion file — pairs with the Character Bible and Acting Guide. Use this to keep each character SOUNDING the same across every future video.)

---

## WHY THIS MATTERS FOR AUTOMATION

Look consistency + movement consistency isn't complete without VOICE consistency. This file locks each character's vocal identity — pitch, tone, pacing, accent flavor, and speech quirks — so whichever TTS/voice tool you use (ElevenLabs, Veo, etc.), you can feed the same voice description every time and get a recognizable, repeatable voice.

For Hindi dialogue delivery specifically, this also notes tone/accent guidance so voice generation stays natural and age-appropriate for children's content.

---

## HOW TO USE THIS

1. When generating dialogue audio (TTS) or writing voice notes into Phase 6 video prompts, copy the **LOCKED VOICE PROFILE** line for that character as-is.
2. If using a voice cloning/TTS tool, save each character's profile as a reusable "voice preset" using this description.
3. Never change: pitch range, core tone, or accent flavor. You MAY vary: energy level and pacing slightly based on scene emotion (e.g. Appy can speak faster when panicked, but his pitch/tone stays the same).
