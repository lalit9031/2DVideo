---

## HOW TO USE THIS FOR AUTOMATION

1. This file is your **single source of truth** for character appearance.
2. When generating ANY new video (Phase 3 in the master pipeline), pull the locked description below instead of writing a fresh one.
3. Only vary: **outfit** (if story requires a costume change), **pose**, and **expression** — never the base face, body shape, skin tone, or eye color.
4. File names below are fixed. Reuse them across all scripts so your asset library stays organized (e.g. `AppyHero.png`, `AppyHero_Scene12.png` for variants).
5. If a character appears in a new role (e.g. Appy is hero in Video 1, side character in Video 5), keep the LOCKED DESCRIPTION identical — only the "Role in story" and outfit may change per script.
